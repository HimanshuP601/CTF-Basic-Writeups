## üîç Enumeration

```bash
ls -l /usr/bin/awk
```

Output:
```
-rwsr-xr-x 1 root root 47480 Sep  5  2019 /usr/bin/awk
```

- The `s` in `-rws` indicates it's a SUID binary.
- `awk` runs with **root privileges**.
